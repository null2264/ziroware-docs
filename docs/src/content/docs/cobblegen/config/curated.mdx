### Deepslate Generator

Generate Deepslates below Y=0

```json5
{
  "cobbleGen": [
    {
      "id": "minecraft:cobbled_deepslate",
      "weight": 100,
      "maxY": 0
    },
    {
      "id": "minecraft:cobblestone",
      "weight": 100,
      "minY": -1
    }
  ],
  "stoneGen": [
    {
      "id": "minecraft:deepslate",
      "weight": 100,
      "maxY": 0
    },
    {
      "id": "minecraft:stone",
      "weight": 100,
      "minY": -1
    }
  ],
  "formatVersion": "1.0"
}
```
